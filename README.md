# ひみつ文書

LINEなどで送る文章を、ふたりだけの合言葉で暗号文に変える道具です。
届いた側は同じ合言葉で元の文章にもどせます。

**公開ページ:** https://tad551216-rgb.github.io/himitsu/

「つくる手帖」の余白シリーズのひとつです。
→ https://tad551216-rgb.github.io/tsukuru-techo/yohaku.html

---

## つかいかた

1. 相手と **合言葉** を決めます(LINE以外の方法、できれば口頭で)
2. **送る人:** 本文と合言葉を入れて「暗号文をつくる」→ コピーしてLINEで送信
3. **受け取る人:** 届いた暗号文をコピーしてこのページに貼り、同じ合言葉で「文章にもどす」

LINE上を流れるのは `🔒himitsu1:` で始まる意味のない文字列だけです。
合言葉を知らない人には、誰にも読めません。

### 注意

- 合言葉そのものをLINEで送っては意味がありません。必ず別の経路で共有してください
- 合言葉は長く、推測されにくいものほど安全です
- 暗号文は 🔒 から最後まで丸ごとコピーしてください(途中で欠けると復号できません)

## 安心して使えるように

- **すべて端末内で完結** — 文章も合言葉も、どこにも送信・保存されません
- **オフラインで動作** — 一度開けば、電波のない場所でも使えます
- **ホーム画面に追加可** — iPhoneのSafariで「共有」→「ホーム画面に追加」でアプリのように使えます

## しくみ(技術メモ)

| 項目 | 内容 |
|---|---|
| 暗号方式 | AES-GCM 256bit(改ざん検知つき) |
| 鍵導出 | PBKDF2-SHA256・310,000回(合言葉から導出) |
| ソルト/IV | 暗号化のたびにランダム生成(同じ文章でも毎回違う暗号文になる) |
| 実装 | ブラウザ標準の Web Crypto API のみ。外部ライブラリなし |
| 形式 | `🔒himitsu1:` + Base64(salt 16byte + IV 12byte + 暗号文) |

合言葉が違う場合や暗号文が一文字でも欠けている場合は、
GCMの認証タグにより復号エラーになります(誤った文章が表示されることはありません)。
